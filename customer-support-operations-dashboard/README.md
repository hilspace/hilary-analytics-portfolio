# Customer Support Operations Dashboard

## Overview
This project analyzes customer support ticket data to evaluate workload, issue types, and resolution times. It demonstrates creating calculated fields, pivot tables, charts, and dashboards to provide actionable operational insights.

## Tools Used
- Google Sheets
- Pivot Tables
- Charts and visualizations

## Key Steps
1. Created pivot tables for:
   - Tickets by Status
   - Tickets by Issue Type
   - Average Resolution Time by Issue Type
2. Built a dashboard combining key metrics
3. Added insights for each chart to explain trends

## Key Insights
- Majority of tickets are open, showing some team inefficiency
- Refund requests account for the largest share of tickets
- Some issue categories have significantly longer resolution times, highlighting potential bottlenecks
