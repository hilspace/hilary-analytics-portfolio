# BMW Global Sales Analysis (2018–2025)

## Overview
This project analyzes BMW’s global sales data to identify trends by region, model, and year. It demonstrates the ability to create pivot tables, charts, and dashboards to support business insights.

## Tools Used
- Google Sheets
- Pivot Tables
- Charts and visualizations

## Key Insights
- BMW sales show steady growth performance between 2018-2025 with minor fluctuations. 
- China represents the largest regional market. 
- SUV models dominate overall sales volume.
